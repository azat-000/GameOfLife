## What does getPattern.initialize(this) do?
### getPattern in instanceof World classes returns the reference of the object of type Pattern which has the method with the heading initialize(World world).
### Calling initialize(this) on the returned Pattern type, we give our current Word object reference as argument, which no matter if our world is of Type ArrayWorld, or PackedWorld, initializes the bord using world.set(...). The world now refers to our object where we called the getPatter..... and thus it does the initialization. All with the help of polymorphism.
